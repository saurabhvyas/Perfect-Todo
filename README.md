
What is Perfect-Todo ?

Its a simple , lightweight Todo App with fast swyping actions, swype right to "complete" a todo and left to "remove" a todo.
Its built using Ionic framework .

It uses SqlDatabase Engine .

Platform : Android

Min Android Required : 5.1

See it live : https://play.google.com/store/apps/details?id=com.ionicframework.playground634759&hl=en
